# Bentael Guide Website - Draft 1

Working frontend + backend website prototype based on the project report.

## Included modules

- `backend/` - Express API + static file server
- `frontend/` - Nature-themed single-page web app

## Implemented features (draft)

- Offline-aware UI with sync queue in `localStorage`
- Trail list with distance, ETA, difficulty
- Landmark and wildlife knowledge sections
- Activity tracker input
- Notes/comment/photo log input
- Trail ratings and reviews
- Admin visitor snapshot
- Browser geolocation sender for visitor tracking
- Map placeholder section (to be replaced when map data is provided)

## Run locally

1. Open terminal in `website/backend`
2. Install dependencies:
   - `npm install`
3. Start server:
   - `npm start`
4. Open:
   - `http://localhost:4000`
